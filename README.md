## ✨ 참고사항

출처: https://archive.ics.uci.edu/ml/datasets/statlog+(german+credit+data)

이 데이터세트에는 고객의 신용 위험도를 예측하는 데 사용할 수 있는 10가지 인자가 포함되어 있다.

- Age: 나이
- Sex: 성별
- Job: 직업
- Housing: 주택 소유
- Saving accounts: 예금 계좌 현황
- Checking account: 당좌 계좌 현황
- Credit amount: 채권 금액
- Duration: 기간
- Purpose: 목적
- Risk: 위험도
